Remote OWASP ZAP Daemon with Flask Integration

This project features a custom interface for managing OWASP ZAP (Zed Attack Proxy) as a remote daemon, built with Flask for both frontend and backend operations. Real-time updates during scans are provided using WebSockets, allowing users to monitor scan progress and view live data as vulnerabilities are detected.
Scan reports are stored in an SQLite database for future reference, and users can generate comprehensive .pdf reports on demand. This tool is designed to enhance the efficiency and accessibility of automated security scans, making it valuable for ongoing vulnerability management.
