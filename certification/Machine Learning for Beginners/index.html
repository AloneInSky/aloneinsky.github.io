This class is designed for beginners who want to start a career in Machine Learning while adhering to the latest industry competency standards. Upon completing this class, students will have the skills to develop machine learning projects focused on classification, regression, and clustering using tabular data.

Materials Covered

Module 1: Hi, Machine Learning

Introduces the fundamentals of machine learning, including its definition, history, and applications across various fields.

Module 2: Machine Learning Workflow

Explains the workflow in a machine learning project, from data collection to model evaluation and deployment.

Module 3: Supervised Learning - Classification

Discusses classification concepts in supervised learning and commonly used algorithms such as KNN and Decision Tree.

Module 4: Supervised Learning - Regression

Covers regression techniques for predicting continuous values, including linear regression and model evaluation.

Module 5: Unsupervised Learning - Clustering

Introduces clustering methods for grouping unlabeled data using algorithms like K-Means.

Module 6: Feature Engineering Techniques

Explores strategies in feature engineering to optimize features and enhance model performance.

Module 7: Overfitting and Underfitting

Explains the concepts of overfitting and underfitting in machine learning models and techniques to prevent them.

Module 8: Model Optimization with Hyperparameter Tuning

Examines the importance of hyperparameter tuning and techniques used to improve model accuracy.

Learning Evaluation

Final Submission: Project Development of Supervised and Unsupervised Learning Models on Tabular Data

Building a machine learning model using various supervised and unsupervised learning methods on real-world cases.
