This class is intended for machine learning developers who want to explore applied machine learning across various domain, aligned with the latest industry competency standards.

Materials Covered

Machine Learning System Design

Learn the stages of building a machine learning system and understand system design principles.

Building a Machine Learning Project

Understanding the structures of a machine learning project and learn how to communicate and publish it to build a professional portfolio.

Case Study 1: Predictive Analytics

Learn the k-Nearest Neighbor, Random Forest, and AdaBoost algorithms and their application in predictive analytics.

Case Study 2: Sentiment Analysis

Develop sentiment analysis models using Deep Learning techniques and Support Vector Machines.

Case Study 3: Computer Vision

Learn computer vision techniques for image recognition and object detection.

Case Study 4: Recommendation Systems

Study the methods used in recommendation systems and build your own project using those techniques.

Learning Evaluation

Final Exam

A comprehensive test covering all the key concepts and techniques taught throughout the class.

Project Submission: Predictive Analytics

Develop a machine learning model to solve a real-world business or societal problem using predictive analytics, and present the results in a clear project report.

Final Submission: Recommendation System Project

Develop a machine learning model and project report for a recommendation system using approaches such as Content-Based Filtering or Collaborative Filtering.
