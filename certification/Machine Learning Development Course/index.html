This class is designed for Machine Learning Developers who want to learn about computer vision, natural language processing, and deployment according to industry competency standards. After completing the course, students are expected to be capable of building deep learning projects focused on processing and predicting text and image data.

Materials Covered

Introduction to Deep Learning

Delving deeper into the concept of deep learning and the use of more complex neural networks to learn deep data representations.

Neural Networks with TensorFlow and Keras

Learning the practical steps of creating and training neural network models using various leading platforms and libraries.

Natural Language Processing

Implementing practical steps in an NLP project using TensorFlow, including text processing, data labeling, and building models for tasks such as text classification or sentiment analysis.

Time Series

Learning about building deep learning models for time series analysis, including techniques like LSTM (Long Short-Term Memory) and RNN (Recurrent Neural Networks).

Image Classification

Demonstrating how to build and train deep learning models for image classification, including preprocessing techniques and model evaluation.

Recommendation Systems

Reviewing various types of recommendation systems and how to implement them using deep learning to provide more personalized and accurate recommendations.

Reinforcement Learning

Deepening the understanding of reinforcement learning concepts, including algorithms like Markov Model and Q-learning, as well as applications in adaptive decision-making.

Machine Learning Model Conversion

Discussing strategies and techniques for deploying and evaluating machine learning models on various platforms and environments.

The Game Changer: Generative AI

Explaining the concept of generative AI and demonstrating examples of its application in generating new content such as images, text, and multimodal content.

Learning Evaluation

Mid-Term Submission: Sentiment Analysis Project

Demonstrating an understanding of Natural Language Processing (NLP) through sentiment analysis.

Final Submission: Development of a Deep Learning Model for Image Classification

Creating a deep learning model capable of performing image classification with real-world applications.
